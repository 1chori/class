지금까지 배운 내용 태그 : div, p, a, br, span, h1~h6, img, table 이번 시간엔 스타일을 주자. 속성(attribute) style을 주자
style 속성 - CSS(Cascading style sheets)
width : 너비, max-width, min-width
height : 높이, max-height, min-height
background-color : 배경색, 색상을 태그에 입힐 수 있다 padding(태그 내부 여백), margin(외부의 여백) : top, right, bottom, left 순서로 입력 값을 하나만 쓰면 4방면에 적용된다(ex. margin : 0;) 값을 두개 쓰면 상하, 좌우 값을 적용할 수 있다 box-sizing : 박스의 크기를 어떻게 설정할지 기준을 설정
속성을 추가 할 때 속성 명 = '속성 값' display : 영역의 노출 크기를 어떻게 설정할지 기준을 설정
